Confidence Interval Estimates‚Äã

  • CI is a range of values that is likely to include the population parameter with a certain level of confidence.
  • Has a lower limit (LL) and an upper limit (UL).
  • Takes into consideration the variation in sample statistics from sample to sample.
  • Examples:
    • 95% CI for ùúá implies that 95% of all confidence intervals, constructed using all possible samples will contain the population mean.
    • 99% CI for ùúá implies that 99% of all confidence intervals, constructed using all possible samples will contain the population mean.
